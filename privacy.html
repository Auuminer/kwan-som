Privacy Policy

Kwan & Som · Last updated: June 11, 2026

This Privacy Policy explains how Kwan & Som ("we", "our", or "us") collects, uses, and protects information when you use our TikTok API integration ("the App").

1. Information We Access

Our application requests the following TikTok API scopes. Each scope is requested for a specific, limited purpose:

user.info.basic — Public profile information (display name, avatar, username)

Purpose: To identify the authenticated account within the App interface and personalize the creator experience. This data is displayed only to the authenticated user.

Storage: Not stored permanently. Held in memory during the active session only.

user.info.stats — Account statistics (follower count, engagement metrics)

Purpose: To provide creators with basic content performance feedback within the App, helping them understand their audience without leaving the publishing workflow.

Storage: Not stored. Read in real-time and displayed to the authenticated user only.

video.upload — Upload video files to TikTok servers

Purpose: Core App functionality — upload AI-generated animated content to TikTok for publishing. Videos are processed in-memory and uploaded directly.

Storage: No video files are stored on our servers. Upload is a direct pass-through operation.

video.publish — Publish uploaded videos to the creator's profile

Purpose: Core App functionality — publish uploaded videos as public posts on the creator's TikTok account. Publication only occurs with explicit creator authorization.

Storage: No publishing records are stored. Each publish action is a one-time API call.

2. How We Use Information

All data accessed through TikTok API scopes is used solely within the scope of providing the content publishing service. Specifically:

  • To display the connected account name within the App interface
  • To upload and publish animated video content as authorized by the creator
  • To provide basic engagement statistics for content performance feedback

3. Data Storage & Retention

  • No permanent storage: We do not store user profile data, video content, or publishing records on any server or database.
  • Session-only tokens: OAuth access tokens are held in memory during the active session and discarded upon logout, token expiration, or app closure.
  • No logs: We do not maintain access logs, usage logs, or analytics databases containing user data.

4. Data Sharing

We do not share, sell, rent, trade, or transfer any user data to third parties. No data leaves the App's in-memory session context. We have no advertising partners, analytics providers, or data brokers integrated into the App.

5. Data Security

We implement the following security measures:

  • OAuth 2.0 PKCE: Industry-standard secure authorization flow with code challenge verification
  • HTTPS encryption: All API communication is encrypted in transit via TLS
  • Memory-only tokens: Access credentials are never written to disk, logs, or databases
  • No credential storage: Client secrets are never hardcoded or stored in the application package

6. Your Rights & Control

  • Revoke access anytime: Go to TikTok > Settings > Security > Apps & Sessions, find "Kwan & Som", and tap Remove. All temporary session data is immediately discarded.
  • No account required: The App does not maintain its own user accounts or databases. Your relationship is solely through TikTok OAuth.
  • Transparency: You can see exactly which scopes you're authorizing on TikTok's OAuth consent screen before granting access.

7. Children's Privacy

The App is not directed at children under 13. We do not knowingly collect information from children. The App's content is general-audience pet entertainment.

8. Changes to This Policy

We may update this Privacy Policy. Material changes will be reflected in the "Last updated" date. Continued use after changes constitutes acceptance.
